Описание методов модуля диспетчиризации лифтов

  • http://sarov.us.to:9900/lift/lifts - список лифтов и их данных: id, тип, адрес в диспетчерской, полный адрес, географические координаты
  • http://sarov.us.to:9900/lift/states - состояния всех лифтов: id лифта, timestamp состояния, код ошибки, описания ошибки, состояние (0 - всё хорошо, зелёный; 1 - ошибка, красный; 2 - нет связи с лифтом, жёлтый; 3 - неизвестно, серый). Состояния лифтов обновляются примерно 1 раз в минуту.
  • http://sarov.us.to:9900/lift/states/errors - список лифтов с ошибками
  • http://sarov.us.to:9900/lift/states/problems - список лифтов с проблемами
  • http://sarov.us.to:9900/lift/events/{id} - все события лифта с id = {id}. Их по каждому лифту довольно много, до 4000 доходит у некоторых, многие неинформативны. События с 2014 года. Например, http://sarov.us.to:9900/lift/events/22
  • http://sarov.us.to:9900/lift/events/{id}/{count} - последние события лифта с id = {id} в количестве {count}, упорядоченные по дате-времени в обратном порядке.
  • http://sarov.us.to:9900/lift/alert - список лифтов с появившимися ошибками. Для ситуационного окна.
